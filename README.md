# 📝 Formulário de Matrícula - School of Rock

![Status do Projeto](https://img.shields.io/badge/Status-Conclu%C3%ADdo-brightgreen)
![Tecnologias](https://img.shields.io/badge/Stack-HTML5%20%2F%20CSS3-blue)

Este projeto é um **formulário de matrícula** moderno e funcional desenvolvido para a "Escola Estrelas do Amanhã". O foco principal foi a criação de uma interface limpa, utilizando semântica HTML e uma arquitetura CSS modular para garantir facilidade de manutenção e escalabilidade.

---

## 🚀 Tecnologias Utilizadas

* **HTML5**: Estruturação semântica do formulário, utilizando grupos de campos (`fieldset`) e legendas para melhor acessibilidade.
* **CSS3**: Estilização avançada com foco em:
    * **Variáveis CSS**: Centralização de cores e fontes para consistência visual.
    * **Flexbox & Grid**: Alinhamento preciso dos elementos do formulário.
    * **Customização de Inputs**: Estilização personalizada de campos de texto, seletores e botões.

---

## 🧩 Estrutura do Projeto

O projeto segue uma organização de arquivos profissional, separando as responsabilidades de estilização:

```text
├── index.html          # Estrutura principal do formulário
├── assets/             # Ícones e imagens do projeto
└── styles/             # Pasta contendo os módulos CSS
    ├── index.css       # Arquivo de entrada (imports)
    ├── global.css      # Estilos globais e variáveis
    ├── header.css      # Estilização do cabeçalho
    ├── form.css        # Regras específicas para os campos do formulário
    └── utilities.css   # Classes utilitárias (espaçamentos, fontes)
